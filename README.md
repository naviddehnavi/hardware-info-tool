# Hardware Information Tool

This project displays detailed information about the hardware of the Windows system it runs on. The script shows CPU, memory, GPU, disk usage, motherboard details, operating system data and network interfaces.

## Requirements

- Windows 10 or 11
- Python 3.8 or newer

## Usage

1. Ensure Python is installed and available in your PATH.
2. Double-click `start.bat` to launch the tool. The batch file checks for required Python packages and installs them automatically if missing. It then runs `main.py` and displays the information in the console.

No additional command-line steps are required.
